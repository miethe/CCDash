# CCDash — Wiki Spec

## Domain

CCDash is a local-first agentic project dashboard and analytics platform that bridges traditional project management with AI development forensics, providing traceability from tasks and commits back to specific agent sessions with token-cost and workflow-quality analytics.

## Topics

- Agent session forensics: thought process introspection, tool usage, subagent topology, cost breakdowns
- Workflow analytics: identifying high-performing vs. wasteful agent workflows across projects
- Data integration: Git history, markdown frontmatter, Claude Code and Codex session log parsing

## Conventions

- Hand-curate: naming, tagging, and routing conventions for this project's artifacts
